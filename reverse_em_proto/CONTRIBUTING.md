## Core Principles

- Stewardship over ownership.
- Transparency over obscurity.
- Protection of children, communities, and the planet.
- No in‑between: all contributions must align with dignity and universal care.

## How to Contribute

1. Fork the repository.
2. Create a feature branch with a clear name.
3. Document your changes thoroughly.
4. Submit a pull request describing:
   - What you changed
   - Why you changed it
   - How it aligns with Space Leaf Corp stewardship principles

## Code of Conduct

Contributors must:

- Treat all collaborators with respect.
- Avoid harmful, exploitative, or deceptive behavior.
- Uphold the No In‑Between Clause.
